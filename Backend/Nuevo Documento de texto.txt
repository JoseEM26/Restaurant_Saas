Microservicios (15 módulos)
Infraestructura: config-server, api-gateway, eureka-server, common (librería compartida)
Negocio: ms-auth-security, ms-core-maestros, ms-almacen, ms-compras, ms-finanzas, ms-contabilidad, ms-rrhh, ms-activos-fijos, ms-produccion, ms-ventas
Soporte: ms-auditoria, ms-reportes, ms-notificaciones, ms-worker
Patrón por microservicio
Controller → Service → Repository (JPA), con DTOs mapeados por MapStruct, validación con Bean Validation, multi-tenant (vi TenantContext), y comunicación entre servicios vía Feign (ej. ms-finanzas validando plan contable contra ms-core-maestros).

Resumen de una línea: Angular 20 + Ionic/Capacitor + Tailwind + AG Grid en el front; Java 21 + Spring Boot 3.5 + Spring Cloud (Gateway/Eureka/Config) con PostgreSQL, Feign, RabbitMQ y JWT en el back.
